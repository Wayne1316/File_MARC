File_MARC package for PHP

Hey, welcome to the File_MARC package for PHP. If you don't know what MARC is, then you probably don't need this package -- but you can find out more about MARC at Library of Congress: MARC standards.

Requirements

  • PHP 5: this package was primarily developed with PHP 5.1.4. I have successfully tested 5.2.0RC1 through RC4.
  • Structures_LinkedList package, also under development

Download

Current release: current.tgz.

See Previous releases for older versions of this code.

Installation

  1. Download the File_MARC package.
  2. Install the Structures_LinkedList prerequisite package (currently in alpha) with the PEAR installer: 
    # pear install Structures_LinkedList-alpha
  3. Forcefully install the File_MARC package with the PEAR installer: 
    # pear install --force File_MARC-0.0.#.tgz

Future

More documentation, more MARC-oriented functionality. I plan to make MARC_Record->appendField() append the field to the end of the corresponding [0-9]xx range to keep records conformant with LOC guidelines. There's no way of sorting the MARC fields... that probably needs to happen. If I'm really aggressive, I might try adding levels of MARC standards compliance for error modes (e.g. only one 1XX field per bibliographic record). Or adding semantic shortcuts for a given record based on the fixed fields values (e.g. $record->getLanguage() or the like).
